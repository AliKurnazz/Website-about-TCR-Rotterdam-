Software Development

Een softwareontwikkelaar is iemand die zich bezighoudt met het programmeren van software. Dit kan een bedrijf zijn met softwareontwikkelaars in dienst of één persoon.

Wat is software

Software zijn computerprogramma's. Voorbeelden hiervan zijn PowerPoint, Word en Firefox. Programma's op de computer noem je software. Er zijn programma's waarmee je van alles kunt doen. Steeds meer software kun je ook op internet vinden.Programma's op de computer noem je software.
